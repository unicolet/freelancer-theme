Unprecedented visibility

Know in near real time what is happening inside your middleware(s): messages, events, channel statistics, errors. Extract daily statistics from the embedded http engine to implement proactive monitoring with any tool.

Instant search

Middleware Insight uses the award winning Elasticsearch engine for storing data and can scale vertically and horizontally to support any load.

Searching and aggregating across days of data is instantaneous and Middleware Insight can help reduce the load on the middleware persistence layer.

Drill down to the exact raw content of a single message or channel with a few mouse clicks.

BYOD

Bring your own dashboard.

Build dashboards for different users: helpdesk, IT, analysts.

Middleware Insight data is integration friendly and can be aggregated with other system metrics (cpu, memory, etc) to build multi-faceted dashboards for Ops.
